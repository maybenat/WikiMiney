# WikiMiney

### Dependencies
* [RVM](http://rvm.io)


### Getting started with this Rails App

```
git clone https://github.com/nataliemcmullen/WikiMiney.git wikiminey
cd wikiminey
rvm use --create --rvmrc 2.2@wikiminey
rvm rvmrc warning ignore '/path/to/wikiminey/.rvmrc'
bundle install
rails server
```
